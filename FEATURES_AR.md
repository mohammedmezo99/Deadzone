# ميزات موقع ProjectMove

## 🌟 الميزات العامة

### 1. **الصفحة الرئيسية**
- قسم Hero متميز مع **تنبيهات البث الديناميكية**
- إبراز ميزات عالية التأثير مع رسوم متحركة عند التمرير
- **إحصائيات ديناميكية**: عدادات مباشرة للمستخدمين النشطين (50K+) وعدد الأجهزة مع رسوم متحركة
- محتوى ديناميكي محدث فورياً (لا مزيد من البيانات القديمة)

### 2. **صفحة الميزات**
- قائمة شاملة بميزات DeadZone
- بطاقات ميزات تفاعلية
- عروض توضيحية بصرية
- واجهة مستخدم متميزة مع رسوم متحركة

### 3. **مركز التنزيل**
- واجهة اختيار الجهاز
- تصفح إصدارات ROM
- روابط تنزيل مباشرة
- معلومات البناء (التاريخ، الحجم، سجل التغييرات)
- الوصول إلى الأرشيف للإصدارات القديمة

### 4. **دليل التثبيت**
- تعليمات التثبيت خطوة بخطوة
- قائمة فحص المتطلبات
- قسم استكشاف الأخطاء وإصلاحها
- إخلاء مسؤولية التحذيرات
- ملاحظات خاصة بالجهاز

### 5. **نظام الأسئلة الشائعة (FAQ)**
- محتوى ديناميكي من قاعدة البيانات
- وظيفة البحث
- تصفية حسب الفئة
- واجهة أكورديون
- سهل التصفح

### 6. **الاتصال والدعم**
- نموذج اتصال مع 3 أنواع:
  - استفسارات عامة
  - تقارير الأخطاء
  - طلبات الميزات
- التحقق من صحة النموذج
- تأكيد النجاح
- إرسال مباشر إلى لوحة الإدارة

### 7. **معرض لقطات الشاشة**
- شبكة صور متجاوبة
- تصفية حسب الفئة
- عرض Lightbox للصور بالحجم الكامل
- تحميل محسّن
- تخطيط متوافق مع الهواتف المحمولة

### 8. **صفحة المجتمع**
- روابط وسائل التواصل الاجتماعي الديناميكية
- أيقونات المنصات (GitHub، Telegram، Discord، Twitter)
- إرشادات المجتمع
- دعوات للانضمام

### 9. **صفحة الفريق**
- ملفات تعريف أعضاء الفريق
- مؤشرات الأدوار
- معلومات البلد
- روابط التواصل الاجتماعي (Twitter، موقع الويب)
- شارات المؤسس/المطور

### 10. **صفحة حول**
- معلومات المشروع
- بيان المهمة
- تاريخ المشروع

### 11. **الصفحات القانونية**
- سياسة الخصوصية
- شروط الخدمة

### 13. **مزامنة البيانات الفورية** 🔄
- **إعادة التحقق التلقائي**: تعديلات المسؤول تقوم بتطهير ذاكرة التخزين المؤقت العامة تلقائياً
- **التوجيه الديناميكي**: البيانات الهامة محدثة دائماً من قاعدة البيانات (Force Dynamic)
- **تحديث بدون تأخير**: التغييرات تظهر في المعرض/الأسئلة الشائعة/التواصل الاجتماعي فورياً

## ⚙️ ميزات لوحة الإدارة

### المصادقة
- نظام تسجيل دخول آمن
- إدارة الجلسات
- مسارات إدارة محمية

### 1. **نظرة عامة على لوحة التحكم**
- الإحصائيات والمقاييس
- إجراءات سريعة
- الأنشطة الأخيرة

### 2. **لوحة التحليلات** 📊
- **إحصائيات في الوقت الفعلي:**
  - إجمالي التنزيلات (طوال الوقت)
  - التنزيلات اليوم مع نسبة النمو %
  - التنزيلات هذا الأسبوع
  - التنزيلات هذا الشهر
  - عدد الأجهزة النشطة

- **التحليلات المرئية:**
  - مخطط اتجاهات التنزيل (آخر 30 يومًا)
  - مخطط شريطي للأجهزة الشائعة (أفضل 10)
  - التوزيع الجغرافي (أفضل 20 دولة)
  - تفصيل المدن للدولة الأولى

- **التتبع المحسّن:**
  - عنوان IP والموقع الجغرافي
  - وكيل المستخدم (المتصفح/نظام التشغيل)
  - تحديد الجهاز
  - طوابع زمنية للتنزيل

- **نشاط في الوقت الفعلي**: سجلات النظام المباشرة من نشاط قاعدة البيانات الفعلي
- **تنقل شريط السفلي الأصلي**: قائمة محمول بأسلوب التطبيقات الأصلية مع تأثيرات الضبابية
- **Glassmorphism متميز**: تراكبات شفافة عالية الوضوح في جميع أنحاء لوحة التحكم

### 4. **إدارة الأجهزة**
- إضافة/تعديل/حذف الأجهزة
- مواصفات الجهاز
- إدارة الصور
- تبديل نشط/غير نشط

### 5. **إدارة مجموعات ROM**
- إنشاء إصدارات ROM
- تحميل ملفات ROM
- إضافة سجلات التغييرات
- تعيين تواريخ البناء والأحجام
- أرشفة ROMs القديمة
- ربط ROMs بالأجهزة

### 4. **إدارة الفريق**
- إضافة/تعديل/حذف أعضاء الفريق
- تعيين الأدوار (المؤسس، المطور، الصيانة)
- إضافة روابط التواصل الاجتماعي
- إدارة صور الملف الشخصي
- تعيين البلد

### 5. **إدارة الأسئلة الشائعة**
- إنشاء/تعديل/حذف الأسئلة الشائعة
- تعيين الفئات
- إدارة الترتيب
- تحديثات في الوقت الفعلي
- البحث داخل لوحة الإدارة

### 6. **إدارة نماذج الاتصال**
- عرض جميع الإرساليات
- تصفية حسب النوع (خطأ/ميزة/عام)
- تصفية حسب الحالة (جديد/مقروء/محلول)
- تحديث حالة الإرسال
- قراءة الرسائل

### 7. **إدارة لقطات الشاشة**
- تحميل لقطات الشاشة
- إضافة العناوين والأوصاف
- تعيين الفئات
- إدارة الترتيب
- حذف الصور

### 8. **إدارة الروابط الاجتماعية**
- إضافة منصات التواصل الاجتماعي
- تعيين عناوين URL
- تبديل نشط/غير نشط
- أيقونات خاصة بالمنصة
- تحديثات التذييل في الوقت الفعلي

### 9. **إعدادات النظام**
- تكوين الموقع
- الإعدادات العامة
- تفضيلات المظهر

## 🔧 الميزات التقنية

### الواجهة الأمامية
- **الإطار:** Next.js 14 (App Router)
- **التصميم:** Tailwind CSS
- **الرسوم المتحركة:** Framer Motion
- **الأيقونات:** Lucide React
- **أمان النوع:** TypeScript

### الخادم الخلفي
- **قاعدة البيانات:** PostgreSQL
- **ORM:** Prisma
- **API:** Next.js API Routes (RESTful)
- **المصادقة:** مصادقة آمنة قائمة على الجلسة

### نماذج قاعدة البيانات
- Device (الجهاز)
- Rom
- Download (التنزيل)
- TeamMember (عضو الفريق)
- Faq (الأسئلة الشائعة)
- ContactForm (نموذج الاتصال)
- Screenshot (لقطة الشاشة)
- SocialLink (الرابط الاجتماعي)
- SiteConfig (تكوين الموقع)

### نقاط نهاية API
**عامة:**
- `/api/faq` - الحصول على جميع الأسئلة الشائعة
- `/api/contact` - إرسال نموذج الاتصال
- `/api/screenshots` - الحصول على جميع لقطات الشاشة
- `/api/social` - الحصول على الروابط الاجتماعية النشطة
- `/api/downloads` - الحصول على تنزيلات ROM

**الإدارة:**
- `/api/admin/faq` - CRUD للأسئلة الشائعة
- `/api/admin/contact` - إدارة الإرساليات
- `/api/admin/screenshots` - CRUD للقطات الشاشة
- `/api/admin/social` - CRUD للروابط الاجتماعية
- `/api/admin/devices` - CRUD للأجهزة
- `/api/admin/roms` - CRUD لـ ROMs
- `/api/admin/team` - CRUD لأعضاء الفريق
